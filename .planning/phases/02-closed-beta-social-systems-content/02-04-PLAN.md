**Sales activity feed entry format** (per spec §202):

Each `data.stalls.mailbox.sales` entry renders as: `"[YYYY-MM-DD HH:MM] {buyerName} bought {qty}× {itemName} for {coinsEarned} coins."` Use `os.date("%Y-%m-%d %H:%M", entry.soldAt)` for the timestamp. Use Species.byId[entry.itemId].displayName (or Constants.ITEMS[entry.itemId].displayName) for the item name; fall back to entry.itemId if the lookup fails.

**Recent Sales feed format** (per spec §216):

`"[time-ago] {buyerName} bought {qty}× {itemName} from {sellerName} for {coins} coins."` "time-ago" computed client-side: if soldAt within last 60s → "just now"; within last hour → "Nm ago"; within last day → "Nh ago"; else use os.date.
